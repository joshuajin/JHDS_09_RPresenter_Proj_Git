Shiny App

Here is a Shiny App that demonstrates the potential usefulness of optimizing investment portfolios. The chart shows one scenario of getting greater than 10% of APR for the last ten years from Yahoo Finance historical data.

2 of 5

How It Works

Funds invested will be reviewed and rotated monthly based on the last ten years historical data. User can optionally make the following selections.

  • Picks a portfolio from the available dropdown list
  • Select a performance look-back period, the default value is 3-month
  • Select Top N funds to invest.
  • Select a rank position to keep to keep your funds invested.
  • Click here to play it for yourself.

Under The Hood

Shiny framework for R enables us to turn our analysis into interactive Web Application. You can find source codes of this App here:

  • GUI is implemented in ui.r. User is expected to have minimum knowledge of investment to effectively use the App.
  • Business code logic is implemented in server.r with help of open source packages such as quantmod, systematic investor toolbox.
  • The initial chart needs about 10 seconds to load.Click here to play.

Limitations:
This app is for a demo purpose; it contains three portfolios. You need to modify the source code to add/update/delete portfolios.
